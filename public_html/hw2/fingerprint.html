Fingerprinting Demo (PHP)

This page generates a browser fingerprint in JavaScript and sends it to fp-lookup.php. The server returns a stable uid and sets it as a cookie.

Results

Click the button to run.

Test instructions

  1. Click the button once. You should get a uid.
  2. Clear cookies/site data for bananabread.bar or use a fresh browser profile.
  3. Reload this page and click again. If the fingerprint is stable, you should get the same uid.
